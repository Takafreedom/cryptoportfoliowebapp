【事前準備】

・お使いの無料の Google アカウントを1つ用意するだけでOKです。

1

Google AI Studio にサインイン

設定画面にある「無料の API キー取得はこちら」のリンクをクリックし、Googleアカウントでログインします。

2

「Get API key」をクリック

ログイン後、ダッシュボード画面の左側メニュー上部にある、青い 「Get API key」（または「APIキーを作成」）ボタンを選択します。

3

キーを作成 ＆ コピー

「Create API key」 を押し、任意のGoogle Cloudプロジェクトを選択（無ければその場で新規作成）すると、AI-zaSy... で始まるAPIキーが出力されます。それをコピーして当アプリの入力欄に貼り付ければ、即座にAIリサーチが稼働します！

作成したキーはアプリ内のLocalStorage（あなたのデバイス内）にのみ完全に保護されて保管され、外部の他者サーバーへ送信されることはありません。
